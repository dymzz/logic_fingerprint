# 用户模式快速接入

这页只解决一件事：让你在 3 分钟内把 `logicfp` 接到一个函数上。

如果你没有明确的“单独跑服务”需求，就先只看这页。

## 1. 什么时候直接用用户模式

适合这些情况：

- 你要保护一个普通 Python 函数
- 你要包住 LangChain `invoke()` / `ainvoke()`
- 你要保护 SDK、工具调用、本地脚本任务
- 你不想维护 HTTP 服务、Nginx、handler registrar

这时默认入口就是：

- `from logicfp import protect`
- `from logicfp import create_protector`

## 2. 3 分钟最小接入

先安装：

```bash
pip install logicfp
```

然后直接包住你的函数：

```python
from logicfp import protect


@protect()
def call_model(request):
    return {
        "ok": True,
        "payload": request.payload,
    }
```

调用时传 `payload`：

```python
result = call_model(payload={"text": "hello"})
```

如果你想加输入输出约束：

```python
from pydantic import BaseModel
from logicfp import protect


class ReviewInput(BaseModel):
    text: str


class ReviewOutput(BaseModel):
    summary: str


@protect(input_model=ReviewInput, output_model=ReviewOutput, simple=False)
def review_text(request):
    return {"summary": request.payload["text"][:20]}
```

## 3. 配置放哪里

用户项目里统一放：

```text
your_project/
  config/
    config.yaml
```

最小配置：

```yaml
logicfp:
  instance_id: decorator-node
  default_source: user_function
  backend_type: memory
```

推荐先别上 Redis，先把函数保护链跑通。

## 4. 什么时候用 create_protector()

默认先用 `@protect()`。

只有这些情况，再切到 `create_protector()`：

- 你需要两个不同策略的保护器
- 你想显式指定 `instance_id` 或 `default_source`
- 你想把同一组配置复用到多个函数上

最常见写法：

```python
from logicfp import create_protector


tool_guard = create_protector(
    instance_id="tool-guard",
    default_source="tool_call",
)


@tool_guard.protect(simple=False)
def run_tool(request):
    return {"ok": True}
```

## 5. 常见错误

### 返回值为什么不是原始结果

如果你用了默认写法，`simple=True` 时成功会直接返回结果，失败会抛异常。

如果你想统一拿到 envelope：

```python
@protect(simple=False)
```

### 失败时为什么抛异常

这是 `simple=True` 的默认行为。

如果你要捕获它：

```python
from logicfp.user_mode import ProtectRuntimeError
```

### 配置为什么没生效

先检查两件事：

1. 文件是不是放在 `config/config.yaml`
2. 进程当前工作目录是不是你的项目根目录

如果还不确定，直接打印当前生效配置：

```python
from logicfp.config import DECORATOR_PROFILE, describe_effective_config

print(describe_effective_config(profile=DECORATOR_PROFILE))
```
