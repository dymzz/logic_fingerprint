# 🛡️ 逻辑指纹系统·工程落地执行标准 v1.1

## 一、系统总体架构（分层设计）

### 1.1 理论基式（解释层）
κ_base = β1 c̃ + β2 ṽ + β3 r̃ - β4 h̃_t - β5 h̃_d

用途：
- 离线验证
- A/B 实验
- 可解释分析

### 1.2 部署修正层（抗噪层）
κ_deploy = κ_base - λ_conf * U_conf(t) - λ_cal * U_cal(t)

用途：
- 抑制盲目自信
- 抑制错误自信

### 1.3 安全熔断层（保护层）
κ_final =
    0                    if trigger_failsafe = 1
    κ_deploy             otherwise

---

## 二、数据预处理标准

变量类型与处理：

- 物理/业务量：x̃ = x / S_mean → 限制 [0,3]
- 统计/代理量：x̃ = (x-μ)/σ → 限制 [-3,3]
- 熵：h̃ = H/H_max → 限制 [0,1]

规则：
- 分母为0 → ERR_NORM
- 残差变量按统计量处理

---

## 三、不确定性处理

U_conf(t) = 1 - |2t - 1|^γ   (γ < 1)

U_cal(t) = |PredictedConfidence - HistoricalAccuracy|

降级：
若校准表不可用 → λ_cal = 0
